## Expense Tracker

Keep track of income and expenses. Add and remove items and save to local storage.

## Demo

Live project: https://fabiosimedo.github.io/expense-tracker/

## Improvements in this version

- Expense and income selector, so mobile users do not need to type a negative sign
- Better mobile UX for inputs, buttons and transaction list
- Safer amount handling for expenses and income
- Transactions are still persisted in local storage

## Project Specifications

- Create UI for project
- Display transaction items in DOM
- Show balance, expense and income totals
- Add new transaction and reflect in total
- Delete items from DOM
- Persist to local storage
